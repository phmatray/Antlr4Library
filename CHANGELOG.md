## [1.2.0] - 2025-12-03

### Changed

- Upgraded to .NET 10 and updated NuGet packages (Nuke.Common 9.0.4 to 10.1.0, System.Runtime.Serialization.Formatters 9.0.10 to 10.0.0)
- Upgraded actions/checkout from v4 to v6
- Upgraded actions/upload-artifact from v4 to v5
- Upgraded Antlr4BuildTasks from 12.8.0 to 12.11.0
- Upgraded actions/setup-dotnet from v4 to v5

### Added

- Renovate configuration for automated dependency updates
- Claude PR Assistant and Code Review workflows

## [1.1.0] - 2024-11-30

### Added

- Nuke build system integration
- Support for .NET 9.0
